# Data Validation and Redundancy Removal System

## Project Overview

This project prevents duplicate and redundant data from being stored in a database.

## Features

- Duplicate detection
- Data validation
- Unique record insertion
- Redundancy prevention
- Database efficiency improvement

## Technologies Used

- Python
- Pandas

## How It Works

1. Load existing database.
2. Check new records against existing data.
3. Identify duplicates.
4. Reject redundant entries.
5. Store only unique records.

## Run

```bash
python data_validation.py
```

## Sample Output

```text
✅ Unique and verified data added successfully.

❌ Duplicate ID detected. Record rejected.
```
