# Cloud-Based Bus Pass System

## Project Overview

This project is a simple cloud-based bus pass booking system developed using Python.

## Features

- Online Bus Pass Booking
- Digital Ticket Records
- Fixed Pricing System
- Booking History
- Cloud-Ready Architecture

## Technologies Used

- Python 3

## How to Run

```bash
python bus_pass_system.py
```
