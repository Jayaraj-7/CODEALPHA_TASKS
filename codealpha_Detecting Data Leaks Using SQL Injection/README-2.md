# Detecting Data Leaks Using SQL Injection Protection

## Overview

This project demonstrates how to secure user data against SQL Injection attacks using parameterized SQL queries and encrypted credential storage.

## Features

- Secure login system
- SQL Injection prevention
- Encrypted password storage
- SQLite database
- Lightweight implementation

## Technologies

- Python
- SQLite
- Cryptography

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python secure_login.py
```
